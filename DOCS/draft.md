**Matching Compétences RNCP 37873 / Planning des Cours – Version Révisée**

**BLOC 1 – Développer une application sécurisée**

1. Installer et configurer son environnement de travail en fonction du projet
   Cours liés : AWS Architecture, Azure Architecture
   Analyse : Compétence couverte. Un renforcement est prévu dans les cours cloud concernant la configuration, la chaîne d'outils et les bonnes pratiques DevOps.

2. Développer des interfaces utilisateur
   Cours liés : Full Stack Development – Front, HTML/CSS (E1), JavaScript Fundamentals (E1), Rich Media Content Development (E1)
   Analyse : Compétence couverte dans les modules front-end.

3. Développer des composants métier
   Cours liés : Full Stack Development – Back, Modern Enterprise Software Engineering (Java, C#…)
   Analyse : Compétence couverte dans les enseignements back-end.

4. Contribuer à la gestion d'un projet informatique
   Cours liés : Cours Agile
   Analyse : Compétence couverte. Un renforcement est prévu pour la gestion documentaire et le pilotage.

---

**BLOC 2 – Concevoir et développer une application sécurisée organisée en couches**

5. Analyser les besoins et maquetter une application
   Cours liés : Gestion de projet (E1/E2), UX/UI, Figma
   Analyse : Compétence déjà vue en E1 et E2. Un rappel sera proposé via supports et exemples.

6. Définir l'architecture logicielle d'une application
   Cours liés : Azure Architecture, AWS Architecture, Full Stack Development Back, Full Stack Development Front, Web Programming – PHP Frameworks Advanced (E2), Modern Enterprise Software Engineering (Java/C#), Container Orchestration, Microservices Orchestration, Data Modeling Fundamentals – UML (E1/E2)
   Analyse : Compétence couverte globalement par les cours de développement et de cloud. L'architecture logicielle est abordée dans les parcours front, back, microservices, frameworks avancés et dans les modules cloud.

7. Concevoir et mettre en place une base de données relationnelle
   Cours liés : SQL (E1), BI ETL & Datamart, Functional Design (Merise) E1
   Analyse : Compétence déjà vue en E1/E2. Un rappel SQL/Merise sera mis à disposition sous forme de bibliothèque.

8. Développer des composants d'accès aux données SQL et NoSQL
   Cours liés : NoSQL-MongoDB, BI ETL & Datamart
   Analyse : Compétence couverte pour NoSQL. Un rappel SQL sera prévu dans les cours back-end.

---

**BLOC 3 – Préparer le déploiement d'une application sécurisée**

9. Préparer et exécuter les plans de tests d'une application
   Cours liés : Web Programming – PHP Frameworks Advanced (E2 – Module Testing), Full Stack Development – Back (module Tooling incluant testing), Innovation Project
   Analyse : La compétence est abordée dans plusieurs modules de développement (tests unitaires, intégration, PHPUnit, outils de tests). Un renforcement transversal sera intégré dans Innovation Project pour structurer la démarche : plan de tests, automatisation, intégration continue.

10. Préparer et documenter le déploiement d'une application
    Cours liés : Innovation Project
    Analyse : Renforcement prévu. Des supports complémentaires traiteront la documentation, les procédures de déploiement et les scripts d'automatisation.

11. Contribuer à la mise en production dans une démarche DevOps
    Cours liés : AWS Architecture, Innovation Project
    Analyse : Compétence couverte dans AWS Architecture, conformément à l'intention initiale, avec un renforcement DevOps dans Innovation Project.
